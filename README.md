# Personal AI Digital Twin 🚀

An autonomous, API-driven digital twin built to represent my identity, answer questions about my background, and capture leads while I'm busy building.

**👉 Talk to my AI here:** [https://personal-ai-agent-96aq.onrender.com](https://personal-ai-agent-96aq.onrender.com)

## 🌟 Features
- **Strict Persona Context:** Built directly on top of my real-life resume, LinkedIn, and personal goals. It knows my technical stack, my struggles, and my ambitions.
- **Persistent Memory:** Seamless conversational state management allowing for long, contextual discussions.
- **Lead Capture Protocol:** Automatically detects when a user shows interest in collaborating or hiring, and pivots the conversation to capture contact info.
- **Dual-Storage Routing:** Saves leads locally and instantly pushes notifications to my phone via Telegram.
- **Modern UI:** Premium, responsive, glassmorphic chat interface.

## 🛠️ Tech Stack
- **Backend:** FastAPI, Python 3
- **AI Brain:** Llama 3.3 70B Versatile (via Groq API)
- **Frontend:** Vanilla HTML / CSS / JS
- **Hosting:** Render
